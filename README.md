5. Listing NFT for sale

```
dfx canister call opend listItem '(principal "rdmx6-jaaaa-aaaaa-aaadq-cai",2)'

replace this principal id with minted nft principal id
```

6. Get OpenD canister ID:

```
dfx canister id opend
```

7. Transfer NFT To OpenD

```
dfx canister call <NFT CANISTER ID> transferOwnership '(principal "<OpenD canister id>",true)'

```

6. Head to localhost

http://localhost:8080/
